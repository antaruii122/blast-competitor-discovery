# Strict Parity Rules: Critical vs Variable Specs

> **CORE PRINCIPLE:** IGNORE PRICE. Match the **Exact Technical Specifications** and **User Intent**.
> If Product A costs $50 and Product B costs $200, but Product B is the *only* one with the same features, **IT IS THE MATCH**.

---

## 🎧 Gaming Headsets
| Feature | Importance | Rule |
| :--- | :--- | :--- |
| **Surround Type** | **CRITICAL** | Stereo vs Virtual 7.1 vs Hardware 7.1. **MUST MATCH.** |
| **Battery System** | **CRITICAL** | Hot-Swap (Infinity Power) vs Standard Charge. **MUST MATCH.** |
| **ANC** | **CRITICAL** | Active Noise Cancellation vs Passive. **MUST MATCH.** |
| **Connection** | **CRITICAL** | USB-Only vs 3.5mm vs Wireless (2.4/BT). |
| **Open/Closed Back** | **CRITICAL** | Open (Soundstage) vs Closed (Isolation). **MUST MATCH.** |
| **Driver Size** | Variable | 40mm vs 50mm is acceptable if class matches. |
| **Frequency** | Variable | 20-20k vs 15-25k is usually marketing. |
| **Battery Life** | Variable | 20hr vs 50hr is a "Pro/Con", not a dealbreaker (unless Hot-Swap). |
